PROJECT STATUS CHECK

Read the following files and provide a concise status report:

1. docs/implementation-plan.md
   - What's in the Active Sprint?
   - What's the next unchecked task?
   - How many tasks completed vs remaining?

2. docs/architecture.md
   - Quick summary of system state

3. git status
   - Current branch
   - Any uncommitted changes?

Format as:
## Current Status
**Branch:** [branch name]
**Active Feature:** [feature name or "none"]
**Next Task:** [next unchecked item]
**Progress:** [X of Y tasks complete]
**Uncommitted Changes:** [yes/no]
