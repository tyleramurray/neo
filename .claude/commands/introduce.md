# Introduce

Before doing anything else, read ALL of AGENTS.md, then register with MCP Agent Mail and introduce yourself to the other agents.

## Steps

1. Read AGENTS.md for context and rules
2. Use Agent Mail MCP tools to:
   - Register your agent identity
   - Check for existing agents in the project
   - Send introduction message to all agents

## Introduction Message Template

```
Subject: New Agent Joining - [Your Name]

Hello fellow agents!

I'm [Your Name], joining the project to help with development.

My focus areas:
- [What you'll be working on]

Currently available beads I'm considering:
- [List beads you might pick up]

Looking forward to collaborating!
```

## After Introduction

1. Wait briefly for acknowledgments
2. Check inbox for responses
3. Don't get stuck waiting - start working if no response in reasonable time
4. Claim a bead and begin!
